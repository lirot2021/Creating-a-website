Пошаговое руководство

Создание простого веб-сайта с нуля

1. Подготовка к работе

Установка редактора кода

Для создания сайта нам понадобится текстовый редактор. Рекомендуем использовать VS Code:

  • Скачайте VS Code с официального сайта
  • Установите редактор на компьютер
  • Создайте новую папку для проекта

2. Создание HTML-структуры

Базовая разметка страницы


<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Мой первый сайт</title>
</head>
<body>
    <h1>Привет, мир!</h1>
</body>
</html>

Это базовая структура HTML-документа. Сохраните её в файле index.html

3. Добавление стилей CSS

Создание файла стилей

Создайте файл styles.css и подключите его к HTML:


/* styles.css */
body {
    font-family: Arial, sans-serif;
    margin: 0;
    padding: 20px;
}

h1 {
    color: #007bff;
}

Подключите стили в HTML-файле, добавив в <head>:

<link rel="stylesheet" href="styles.css">
